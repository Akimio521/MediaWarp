# 功能
- [x] 对于协议为Http的Strm文件可以实现302直链播放，流量不经过EmbyServer
  - **推荐配合[AutoFilm](https://github.com/Akimio521/AutoFilm)使用**
  - 已通过测试客户端（Web、iOS Emby、Infuse、Conflux）
